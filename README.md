# TurtleImageChecker
Скрипт позволяющий срисовывать изображения с помощью черепахи

1. Установите библиотеки - напишите в терминале "pip install -r requirements.txt"

2. Вставьте путь к файлу в Image_Path = r"images.webp" заменив images.webp на путь к вашему файлу. (Не рекомендуется использовать папки с названиями на русском, может возникнуть ошибка!)

3. Запустите код и нажмите любую клавишу.
